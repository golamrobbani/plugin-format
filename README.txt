== Installation ==

1. Unzip the downloaded `plugin-format CRM` zip file
2. Upload the 'plugin-format CRM' folder and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate `Plugin Format CRM` from Plugins page

= Requirements =
* **WordPress version:** >= 5.0
* **PHP version:** >= 7.1
